# Greetings API

Manage greetings in different languages.

## GET /api/greetings

Retrieve all greetings.

### Response

Returns an array of greeting objects.

### Example Response

```json
[
  {
    "id": "9ab12c34-5678-90de-f123-456789abcdef",
    "languageId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "greetingText": "Hello",
    "formal": false,
    "createdAt": "2024-01-15T10:00:00Z",
    "updatedAt": "2024-01-15T10:00:00Z"
  }
]
```

### Example Request

```bash
curl http://localhost:8080/api/greetings
```

## GET /api/greetings/{id}

Retrieve a specific greeting by ID.

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the greeting
</ParamField>

### Response

Returns a single greeting object.

### Status Codes

- `200 OK` - Greeting found
- `404 Not Found` - Greeting not found

### Example Request

```bash
curl http://localhost:8080/api/greetings/9ab12c34-5678-90de-f123-456789abcdef
```

---

## GET /api/greetings/by-language/{languageCode}

Retrieve a greeting by language code.

### Path Parameters

<ParamField path="languageCode" type="string" required>
  Language code (e.g., "en", "es", "ko")
</ParamField>

### Query Parameters

<ParamField query="formal" type="boolean">
  Filter by formality (true for formal, false for informal)
</ParamField>

### Response

Returns a greeting with language information.

### Example Response

```json
{
  "language": "English",
  "languageCode": "en",
  "greetingText": "Hello",
  "formal": false
}
```

### Status Codes

- `200 OK` - Greeting found
- `404 Not Found` - No greeting found for language

### Example Requests

```bash
# Get any greeting for English
curl http://localhost:8080/api/greetings/by-language/en

# Get formal greeting for English
curl "http://localhost:8080/api/greetings/by-language/en?formal=true"

# Get informal greeting for Spanish
curl "http://localhost:8080/api/greetings/by-language/es?formal=false"
```

## POST /api/greetings

Create a new greeting.

### Request Body

<ParamField body="languageId" type="string" required>
  UUID of the language
</ParamField>

<ParamField body="greetingText" type="string" required>
  The greeting text
</ParamField>

<ParamField body="formal" type="boolean" required>
  Whether this is a formal greeting
</ParamField>

### Example Request

```bash
curl -X POST http://localhost:8080/api/greetings \
  -H "Content-Type: application/json" \
  -d '{
    "languageId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "greetingText": "Good morning",
    "formal": true
  }'
```

### Response

Returns the created greeting object.

### Status Codes

- `201 Created` - Greeting created successfully
- `400 Bad Request` - Invalid request (missing fields or invalid language ID)

---

## PUT /api/greetings/{id}

Update an existing greeting.

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the greeting to update
</ParamField>

### Request Body

<ParamField body="languageId" type="string" required>
  Updated language ID
</ParamField>

<ParamField body="greetingText" type="string" required>
  Updated greeting text
</ParamField>

<ParamField body="formal" type="boolean" required>
  Updated formality flag
</ParamField>

### Example Request

```bash
curl -X PUT http://localhost:8080/api/greetings/9ab12c34-5678-90de-f123-456789abcdef \
  -H "Content-Type: application/json" \
  -d '{
    "languageId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "greetingText": "Hi there",
    "formal": false
  }'
```

### Response

Returns the updated greeting object.

### Status Codes

- `200 OK` - Greeting updated successfully
- `404 Not Found` - Greeting not found
- `400 Bad Request` - Invalid request

## DELETE /api/greetings/{id}

Delete a greeting.

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the greeting to delete
</ParamField>

### Example Request

```bash
curl -X DELETE http://localhost:8080/api/greetings/9ab12c34-5678-90de-f123-456789abcdef
```

### Response

No content returned on success.

### Status Codes

- `204 No Content` - Greeting deleted successfully
- `404 Not Found` - Greeting not found
