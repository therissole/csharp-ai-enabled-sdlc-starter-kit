# csharp-ai-enabled-sdlc-starter-kit

A starter project for C# applications that want to use agentic AI for rapid development.

## Features

This repository is configured with GitHub Copilot instructions to provide intelligent, context-aware assistance for C# development.

### Copilot Configuration

The repository includes:

- **Repository-wide instructions** (`.github/copilot-instructions.md`): General guidelines for C# development, including code style, quality standards, and security practices
- **Custom Copilot agents** in `.github/agents/`:
  - **csharp-dev**: Expert C# developer for writing modern .NET code
  - **test-engineer**: Testing specialist for comprehensive unit and integration tests
  - **docs-writer**: Documentation specialist for maintaining clear technical documentation

These configurations help Copilot provide more relevant, high-quality suggestions tailored to this project's needs.

## Getting Started

This is a starter kit repository. To use it:

1. Clone or fork this repository
2. Add your C# project files to the `src/` directory
3. Add tests to the `tests/` directory
4. Use GitHub Copilot with the configured agents for accelerated development

## Prerequisites

- .NET 8.0 SDK or later
- GitHub Copilot subscription
